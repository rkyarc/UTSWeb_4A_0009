## ⚡ Implementasi Manipulasi DOM
Website ini menerapkan 2 bentuk manipulasi DOM secara dinamis menggunakan jQuery:

1. **Dark Mode / Light Mode Toggle**
   - **Fungsi:** Mengubah tema warna website secara keseluruhan untuk kenyamanan visual pengguna.

2. **Filter Data Katalog Anime**
   - **Fungsi:** Menyaring tayangan anime berdasarkan kategori tertentu (*All*, *School*, atau *Action*) secara langsung tanpa *reload* halaman.
